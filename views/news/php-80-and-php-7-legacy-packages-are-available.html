php パッケージが バージョン 8.0 にアップデートされました。アップストリームの 移行ガイド を参照してください。いくつかのアプリケーションは まだ PHP 8 に対応していないため、バージョン 8 と同時にインストールできる php7 パッケージを提供しています。PHP に依存するパッケージはこの更新を反映し、必要であれば php7 を必要とします。それに従い、 設定を変更する必要があるかもしれません。PHP 7 バイナリと設定は末尾に "7" がついています。

  • /usr/bin/php -> /usr/bin/php7
  • /etc/php -> /etc/php7
  • /usr/bin/php-fpm -> /usr/bin/php-fpm7
  • /usr/lib/systemd/system/php-fpm.service -> /usr/lib/systemd/system/php-fpm7.service
  • /run/php-fpm -> /run/php-fpm7

PHP 7 のためにコンパイルされたサードパーティーのモジュールも提供しています。

  • php7-apcu
  • php7-geoip
  • php7-grpc
  • php7-igbinary
  • php7-imagick
  • php7-memcache
  • php7-memcached
  • php7-mongodb
  • php7-redis

php7 のサポートは 限定されており、ほとんどのアプリケーションが どのくらい早くバージョン 8 に対応するかによりますが、およそ 1 年程度で提供されなくなる可能性が高いことに 注意してください。
