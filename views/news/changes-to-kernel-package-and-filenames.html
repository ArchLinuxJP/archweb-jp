Linux カーネルのバージョン 3.0 が [core] に入ったため、kernel26 パッケージの名前は linux に変更されました。また、カーネルと initrd の名前も以下のように変更されます:

  • vmlinuz26 は vmlinuz-linux になります。
  • kernel26.img は initramfs-linux.img になります。
  • kernel26-fallback.img は initramfs-linux-fallback.img になります。

アップデートによってシステムが壊れないようにするため、ブートローダーの設定が古いファイルを参照していても機能するようにシンボリックリンクが作成されます。ただしブートローダーの設定を更新してシンボリックリンクは削除することを推奨します。
