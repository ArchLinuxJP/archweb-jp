TeXLive 2013 パッケージがすぐに [extra] に入る予定です: TeXLive 2012 からアップグレードしたユーザーは TeX エンジン (latex, xelatex, luatex など) を使おうとすると以下のようなエラーメッセージが表示されるでしょう:

(Fatal format file error; I'm stymied)

pacman によるインストール時に以下のようなメッセージが表示されることでも確認できます:

>>> texlive: recreating all formats...
/usr/bin/fmtutil: line 395: /usr/share/texmf/texconfig/tcfmgr: No such file or directory
fmtutil: config file `fmtutil.cnf' not found.
 done.

/etc/texmf のファイルに変更を加えていない場合、pacman によってインストールされた .pacnew ファイルで設定ファイルを上書きすることで問題を解決できます。pacdiff ユーティリティを使用することで上書きできます。

設定ファイルをマージしたら、以下のコマンドを実行してください:

fmtutil-sys --all > /dev/null

エラーメッセージが返ってこないはずです。何か問題を発見したらバグレポートを送ってください。

TeXLive 2013 をインストールする前に TeXLive 2012 を削除することでも問題を回避できます。
